MoveDex

![MoveDex Screenshot](movedex.png)

A modern Pokémon lookup application that combines live PokéAPI data with curated competitive builds, recommended items, EV spreads, Tera types, and complete level-up movepools.

Live Demo: https://SamuelH98.github.io/MoveDex/

Features
🔍 Search any Pokémon instantly
⚔️ Curated competitive builds for popular Pokémon
📊 Displays recommended:
Item
Ability
Nature
EV Spread
Tera Type
Moveset
📖 Complete level-up learnsets from PokéAPI
🌐 Live Pokémon data fetched in real time
🔗 Automatic Smogon Strategy Dex link for Pokémon without a curated build
📱 Responsive design for desktop and mobile
⚡ Built with pure HTML, CSS, and JavaScript—no frameworks required

Running Locally

Clone the repository:

git clone https://github.com/SamuelH98/MoveDex.git

Open index.html in your browser.

No installation or build process is required.

Deployment

This project is deployed using GitHub Pages.

Push the repository to GitHub.
Go to Settings → Pages.
Set the deployment source to the main branch.
Select the repository root (/) as the folder.
Save the settings.

GitHub Pages will automatically publish the site after a few moments.
